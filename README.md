# Taskly — Full-Stack TODO App

A simple full-stack TODO application built with the MERN stack (MongoDB, Express, React, Node.js).

## Objective

To build a clean, functional, and well-structured web application demonstrating full-stack development capabilities.

## Planned Tech Stack

  **Frontend:** React (Vite)
  **Backend:** Node.js, Express
  **Database:** MongoDB, Mongoose

## Status

Work in Progress. This repository will contain the completed assignment.
